# Login Application

This is a simple React-based login application built with Vite. It includes a login form with validation, a header, and a footer. The application demonstrates basic React concepts such as state management, event handling, and conditional rendering.

## Features

- **Login Form**: Users can log in with a username and password.
- **Validation**: Displays error messages for incorrect login credentials.
- **Reset Functionality**: Allows users to reset the form fields.
- **Conditional Rendering**: Displays different content based on the login state.
- **Responsive Design**: Styled with CSS for a clean and user-friendly interface.
